# 接口文档

ps: baseURL 为 http://localhost:8080

## 1. 用户注册

- 请求地址:`/api/user/sign-up`
- 请求方式:`POST`
- 请求参数：

| 参数名称 | 参数类型 | 是否必须 | 描述 |
| --------| ------- | ------- | ---- |
| username | String | Y | 用户名 |
| password | String | Y | 密码 |
| avatar | String | N | 用户头像 |
| sex | Number | N | 性别 |
| age | Number | N | 年龄 |

- 返回示例:
```js
{
    code: 0, // 0 - 成功， -1 - 失败
    msg：'ok'
}
```
## 2. 用户登录

- 请求地址:`/api/user/sign-in`
- 请求方式:`POST`
- 请求参数：

| 参数名称 | 参数类型 | 是否必须 | 描述 |
| --------| ------- | ------- | ---- |
| username | String | Y | 用户名 |
| password | String | Y | 密码 |

- 返回示例:
```js
{
    code: 0, // 0 - 成功， -1 - 失败
    msg：'ok'
}
```
